Expanding the iOS API

Issued: 2018-10-30

  • Doing things all in code vs. Doing things in Storyboard. How to choose?
  • Hybridizing code and Storyboard:
    instantiateViewController(withIdentifier:), performSegue(withIdentifier:sender:)
  • Segues: present(_:animated:completion:) / dismiss(animated:completion:) on UIViewController vs. pushViewController(_:animated:) / popViewController(animated:) on UINavigationController
  • Color spaces: presenting and creating in terms of HSB but retrieving pixel data as RGB
  • Bit-wise operations: | (OR), & (AND), ~ (NOT), ^ (XOR), << (left-shift), and >> (right-shift); related: exact-width Integer types
  • Drawing off-screen images with an image context
